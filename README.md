# qckMSG

### Komunikator przeznaczony na urządzenia mobilne, ceniący sobie prywatność oraz ochronę danych swoich użytkowników. 

### Wymagany stack technologiczny do realizacji projektu:
* iOS API
* SWIFT
* Cocoa Touch
* MessageKit
* Firebase <Uwierzytelnianie użytkownika, bazy danych>
